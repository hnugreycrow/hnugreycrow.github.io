## 申请友链

欢迎交换友链！如果你也写博客、做技术分享，或者单纯想交个朋友，随时欢迎。

**请按以下格式留言：**

| 字段   | 说明       | 示例                               |
| ------ | ---------- | ---------------------------------- |
| name   | 站点名称   | 某某的博客                         |
| link   | 网站地址   | https://example.com                |
| avatar | 头像链接   | https://example.com/img/avatar.png |
| descr  | 一句话描述 | 记录学习与生活的点滴               |
